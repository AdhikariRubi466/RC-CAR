# RC-CAR

## Overview
This project is an Arduino-based Remote Control (RC) Car developed to demonstrate basic robotics and wireless control concepts. The system allows a user to control the movement of the car using a remote communication method such as Bluetooth or wireless module.

## Features
- Forward and backward movement  
- Left and right turning  
- Motor speed control  
- Wireless communication control  

## Components Used
- Arduino Board (Uno/Nano)
- Motor Driver Module (L298N or similar)
- DC Motors
- Chassis with Wheels
- Bluetooth/Wireless Module
- Battery Pack
- Jumper Wires

## File Description
- `RCCAR.ino` – Main Arduino program for controlling the RC car  
- `RC Car.mp4` – Demonstration video of the working project  

## Working Principle
The Arduino receives control signals from the wireless module and processes them to control the motor driver. Based on the received commands, the motors rotate in specific directions to move the car accordingly.

## Learning Outcome
- Understanding of motor driver interfacing  
- Wireless communication with Arduino  
- Practical implementation of robotics movement control  
- Hands-on experience in hardware assembly and programming
